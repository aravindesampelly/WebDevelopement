Tables in HTML

Table:

Table is a combination of rows and columns which is used to store data in systamatic way easy to understand.

We use table tag for creating table. We use tr tag for creating a row

We use th tag for header row and td for the data

th tag will always be in center of the row cell

we can use colspan and rowspan to arrange the table for our requirement.
